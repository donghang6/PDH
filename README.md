# Fabry-Perot 腔反射与透射系数 MATLAB App

本 App 根据 Eric D. Black, *An introduction to Pound-Drever-Hall laser
frequency stabilization* (2001) 的式 (3.1)，计算对称、无损 Fabry-Perot 腔的
复振幅反射系数与透射系数：

```text
F = r (exp(i phi) - 1) / (1 - r^2 exp(i phi))
T = (1-r^2) exp(i phi/2) / (1 - r^2 exp(i phi))
```

其中 `r` 是单镜振幅反射率，`phi = 2*pi*DeltaNu/FSR`。界面显示：

- 归一化反射强度 `|F|^2` 与反射相位 `arg(F)`
- 归一化透射强度 `|T|^2` 与透射相位 `arg(T)`
- 复平面上的反射、透射系数轨迹
- 无损腔能量守恒校验 `|F|^2 + |T|^2 = 1`
- “PDH 调制”标签页：一阶 EOM 边带、反射光场和探测器拍频
- 自由光谱范围、精细度和腔线宽
- 任意失谐点的数值读数和 CSV 数据导出

## 运行

在 MATLAB 中将当前目录切换到本文件夹，然后运行：

```matlab
launchPDHReflectionApp
```

也可以直接运行：

```matlab
app = PDHReflectionApp;
```

建议使用 MATLAB R2021a 或更新版本。精细度采用文献中的高精细度近似
`Finesse = pi/(1-r^2)`。精确共振时复反射系数为零，因此相位没有定义。

## PDH 调制模型（第一步）

“PDH 调制”标签页采用文献式 (3.2) 的一阶边带近似：

```text
Vpk = Vpp/2
beta = pi*Vpk/Vpi = pi*Vpp/(2*Vpi)
Vpp = 2*beta*Vpi/pi
A0 = E0 J0(beta) F(omega)
A+ = E0 J1(beta) F(omega + Omega)
A- = -E0 J1(beta) F(omega - Omega)
Eref = exp(i omega t) [A0 + A+ exp(i Omega t) + A- exp(-i Omega t)]
Pdet = |A0 + A+ exp(i Omega t) + A- exp(-i Omega t)|^2
```

界面输入调制深度 `beta` 和 EOM 半波电压 `Vpi`，程序实时计算所需的
正弦驱动峰峰值 `Vpp = 2*beta*Vpi/pi`。这个 `Vpp` 指经过射频放大器、
线缆损耗和阻抗终端之后实际到达 EOM 电极的电压；若使用正弦波 RMS
电压，则 `Vrms = Vpp/(2*sqrt(2))`。默认 `Vpi = 5 V`、`beta = 0.30 rad`，
对应 `Vpp = 0.95493 V`。

界面显示反射光场慢变包络、探测器功率、载波/边带功率以及 `f_m`、
`2f_m` 拍频分量。默认设置 `P0 = 1 mW`，此时以 mW 表示的功率数值与
相对于 `P0` 的归一化数值相同。后续子标签页依次完成带通、混频和低通，
最终生成 PDH 误差信号。

### 带通滤波

“带通滤波”子标签页对探测器功率信号执行零相位 Butterworth 型频域滤波。
可设置中心频率、3 dB 带宽、滤波器阶数、采样率和时间窗口。默认通带以
`f_m` 为中心，用于保留载波与边带产生的 `f_m` 拍频，同时抑制直流和
边带之间产生的 `2f_m` 拍频。

滤波页显示：

- 滤波前探测器功率
- 滤波后的交流拍频分量
- 滤波前后单边幅度谱
- 滤波器幅频响应及 `|H(f_m)|`、`|H(2f_m)|`

带通输出已经去除直流，因此可能出现负值；它代表与光电流成比例的交流
分量，不再表示非负的瞬时光功率。当前滤波器用于离线理论分析，不包含
真实电子滤波器的因果相位延迟。顶部两张波形图以无量纲调制周期 `f_m*t`
为横轴，并沿用“反射拍频”页的“显示周期数”，避免在高调制频率下显示
过多周期。

### 混频

“混频”子标签页将带通滤波器的输出与本地振荡器相乘：

```text
VLO(t) = (VLO,pp/2) * sin(2*pi*fLO*t + phiLO)
Vmix(t) = PBPF(t) * VLO(t)
```

可设置 LO 频率、相位和峰峰值 `VLO,pp`，也可一键将 LO 频率同步到当前
调制频率 `f_m`。计算中正弦波峰值为 `VLO,pp/2`；默认 `VLO,pp = 2 V`，
因此峰值为 `1 V`。界面显示带通输入、LO 波形、未低通的混频输出及其单边幅度
谱，并标出差频 `|fLO-f_m|` 与和频 `fLO+f_m`。当 `fLO=f_m` 时，输出包含
直流项和约 `2f_m` 的和频项。“混频”页本身展示未低通的结果，下一子标签页
继续执行低通滤波。当前未模拟真实混频器的转换增益、饱和与附加噪声。
正弦与余弦本振只相差 `90 deg` 相位；仍可通过 LO 相位设置选择最佳解调
正交分量。

### 低通滤波与误差信号

“低通与误差信号”子标签页对混频输出执行零相位 Butterworth 型低通滤波：

```text
Verr(t) = LPF{Vmix(t)}
|HLP(f)| = 1/sqrt(1 + (f/fc)^(2N))
```

PID 目标带宽默认为 `100 kHz`。为了避免在环路带宽处产生明显衰减，低通
截止频率默认设置为 `500 kHz`，也可通过按钮自动设置为 PID 带宽的 5 倍。
页面显示低通前后的时域信号、以 `Delta f/FSR` 为横轴的 PDH 误差曲线、
低通幅频响应、当前误差信号以及共振点斜率。

误差曲线采用当前 LO 相位计算。对于正弦本振，改变 LO 相位会选择不同的
解调正交分量并改变误差曲线的幅度与符号。当前使用零相位离线滤波，尚未
包含真实低通滤波器的相位延迟，也未计算完整 PID 环路的相位裕度。

## 完整链路标签页

“完整链路”顶层标签页将全部步骤组织为方框和箭头：

```text
激光器 -> EOM -> FP 腔反射 -> 光电探测器
                                 |
误差信号 <- 低通/PID <- 混频 <- 带通滤波
```

每个方框内提供该级需要修改的参数，包括激光功率与载波失谐、EOM 的
`Vpi`、调制深度 `beta` 与调制频率（`Vpp` 由程序计算）、
腔长与单镜光强反射率 `R`（振幅反射率 `r=sqrt(R)` 作为显示值）、
探测器带宽、带通参数、LO 参数和低通/PID 带宽。激光场采用
`|E0|^2=P0` 归一化，因此最终误差信号和共振点斜率均随激光功率 `P0`
线性变化。探测器采用一阶电学低通近似，默认 3 dB 带宽为 `400 MHz`。
页面底部直接输出以 `Delta f/FSR` 为横轴的最终 PDH 误差信号，并显示
当前误差、共振点斜率和建议的 P 极性。

建议的 P 极性假设“正控制电压使激光频率升高”。真实系统若执行器调谐
方向相反，PID 的整体极性也应反转。
